# Actions

Reusable action scripts for AI agents. Each file is a self-contained instruction set for a specific repeatable task.

## Format
- `NNN-title.md` - One action per file
- Written as direct instructions to an AI agent
- Should be unambiguous, safe, and reproducible

## Usage
Tell the AI: "Run action `.actions/NNN-title.md`" and it will execute the steps inside.
