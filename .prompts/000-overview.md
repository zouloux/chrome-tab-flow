# Prompts Archive

Each file in this directory contains the full prompt given to an LLM session for this project. Files are numbered sequentially.

## Format
- `NNN-title.md` - Full prompt text, with context about what was asked
- Title reflects the session goal
- Used to replay or continue work in a new session

## Usage
When starting a new session, reference the relevant prompt file so the LLM has full context of what was previously requested.
