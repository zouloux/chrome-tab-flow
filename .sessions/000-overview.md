# Session Summaries

Compact summaries of completed LLM sessions. Max 10 lines each. Used to quickly bring a new LLM instance up to speed.

## Format
- `NNN-title.md` - Summary of one session
- First line: link to the corresponding prompt in `.prompts/`
- Contains: actions taken, files created/modified, key decisions, blockers

## Usage
Read the latest session file before starting new work to understand current project state.
