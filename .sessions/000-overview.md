# Session Summaries

Compact summaries of completed LLM sessions. Each file contains the full prompt and a summary of what was done.

## Format

Each file uses the following structure:

```
---
Session NNN
{Title}
---
# PROMPT
Full prompt given to the LLM for this session.
---
# SESSION
Actions taken, files created/modified, key decisions, and current state.
```

## Usage

Read the latest session file before starting new work to understand current project state.
When starting a new session, write a new `NNN-title.md` file following the format above.
